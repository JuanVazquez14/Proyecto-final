# Proyecto final

Este proyecto es la culminacion de la primera unidad, creado a traves de html.

<h3> Objetivo de la página:</h3>
El objetivo de la página es simular un recetario donde se lleva a cabo una receta para preparar un omelette de huevo, en este caso particular se omitió el uso de recursos como botones y elementos típicos de los formularios debido a que considere que no eran necesarios.
